# Portfolio App
> It's a simple basic Portfolio website created using Node JS, Express JS and MySQL. You will perform CRUD operations with projects data.

## Technologies Used
1.  NodeJS
2.  ExpressJS
3.  MySQL
4.  Handlebars
5.  Foundation UI
6.  Multer

## Prerequisites
- Git
- NodeJS
- CLI
- XAMPP
- MySQL
- NPM

## Installation

##### Clone the latest Repository

```bash
git clone https://github.com/ajinkyacodes/nodejs-apps.git
```

##### Into the project directory

```bash
cd portfolio
```

##### Create MySQL Database

```bash
Create Database `portfolio` with table `projects`
```

Fields for `projects`
- id
- title
- description
- client
- service
- url
- image
- date
- create_date

##### Installing NPM dependencies

```bash
npm install
```

##### Then simply start your app

```bash
node app
```

#### The Server should now be running at http://localhost:3000/
